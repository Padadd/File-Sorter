# 📂 File Sorter

A simple yet powerful utility to automatically organize files in a directory into subfolders based on their file type.

---

## 🚀 Features

- **Automatic Sorting** – Scans a directory and moves files into folders like `Images/`, `Documents/`, `Text/`, and more.
- **User-Friendly CLI** – Run the script and enter a directory path (or press Enter to use the current folder).
- **Graphical Interface** – Launch `gui_scheduler.py` for a simple GUI to select a target directory and manage folders.
- **Automated Scheduling** – Schedule the script to run automatically (daily or weekly) using Windows Task Scheduler.
- **Persistent Settings** – The GUI saves your preferences to `scheduler_settings.json` for future use.
